# 📊 Hospital Emergency Room Dashboard | Microsoft Excel

An interactive **Hospital Emergency Room Dashboard** built using **Microsoft Excel** to monitor patient flow, waiting time, department referrals, admission status, and overall emergency room performance. The dashboard transforms raw hospital data into meaningful insights using Pivot Tables, Pivot Charts, Power Query, and Excel formulas.

---

# 📌 Project Overview

This project analyzes emergency room data to help hospitals monitor patient visits, improve operational efficiency, and support data-driven healthcare decisions.

The dashboard provides insights into patient demographics, admission status, referral departments, waiting time, and daily patient trends.
